# Hey Hood

A citizen empowerment and localized public grievance reporting app with AI-powered moderation, routing, deduplication, and escalation.

## Live Agent Endpoints

* **Text Polish:** `https://hey-hood-agent-text-polish-5n2p.vercel.app`
* **Duplicate Detection:** `https://hey-hood-agent-duplicate-detection.vercel.app`
* **Issue Routing:** `https://hey-hood-agent-issue-routing.vercel.app`
* **Escalation:** `https://hey-hood-agent-escalation.vercel.app`
* **Fake News:** `https://hey-hood-agent-fake-news.vercel.app`
* **Wish Matching:** `https://hey-hood-agent-wish-matching.vercel.app`
